# Poker Help

Калькулятор эквити и EV для покера на Vue 3 + Vite. Учитываются рука, флоп, банк, ставка, количество игроков в банке и позиция.

## Запуск

```bash
npm install
npm run dev
```

Если `npm install` падает из-за пути с кириллицей — склонируйте или скопируйте проект в каталог с путём только из латиницы (например `~/projects/poker-help`) и выполните команды там.

## Как пользоваться

1. **Моя рука** — выберите две карты из выпадающих списков.
2. **Флоп** — выберите три карты флопа.
3. **Банк и ставки** — введите текущий банк, вашу ставку/колл и число игроков в банке (включая вас).
4. **Позиция** — выберите вашу позицию (UTG, MP, CO, BTN, SB, BB).
5. Нажмите **Рассчитать**.

В блоке **Результат** отображаются:
- **Шанс на победу** — процент эквити (круговой индикатор).
- **EV** — ожидаемая прибыль от решения; подпись «Плюсовое решение» или «Минусовое решение».

Стилистика: тёмная тема, карточки-панели, акцентный оранжевый, зелёный/красный для плюса/минуса EV.
